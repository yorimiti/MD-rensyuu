# コード

- バッククォートは、[shift]+[@]キー
- バッククォート３つの後ろに、言語を指定できる
  - cならc言語
  - java
  - html
- スペースがなければ戻る
```cs
function x(){
  Debug.Log("hello.");
}
```

- インラインはバッククォート１つで囲う。`Debug.Log()`はデバッグ表示
